
```css

  linear-gradient(...),
  url('https://your-site.com/image.jpg') center/cover no-repeat;
```

## üí° Background Image Tips

### For Best Results:
1. **High Resolution**: Use at least 1920x1080px
2. **Tech-Themed**: Choose technology/business related images
3. **Not Too Busy**: Simple backgrounds work best
4. **Good Contrast**: Ensure text remains readable
5. **Consistent Theme**: Keep similar color tones throughout

### Adjust Overlay Darkness:
Change the `rgba` values in the gradient:
```css
/* Darker overlay */
linear-gradient(135deg, rgba(15, 23, 42, 0.95) 0%, ...)

/* Lighter overlay */
linear-gradient(135deg, rgba(15, 23, 42, 0.70) 0%, ...)
```
